# Loondry - Laundry Shop Web Application

A modern laundry shop web application built with HTML, CSS, and JavaScript. Loondry provides an easy-to-use interface for managing laundry services, booking appointments, and tracking orders.

## 🎯 Features

- Clean and intuitive user interface
- Service booking system
- Order management
- Responsive design
- Fast and reliable performance

## 🚀 Live Demo

Check out the live application here: https://johnwick944.github.io/looondry/

## 💻 Built With

- HTML5
- CSS3
- JavaScript

## 📝 About

Loondry is a web-based laundry shop management application designed to streamline customer interactions and service delivery. Whether you're looking for a laundry management system or a customer-facing booking platform, Loondry provides a solid foundation.
